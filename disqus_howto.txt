How to display comment count

1.  Place the following code before your site's closing </body> tag:

<script id="dsq-count-scr" src="//rustybear.disqus.com/count.js" async></script>

2.  Append #disqus_thread to the href attribute in your links. This will tell
Disqus which links to look up and return the comment count. For example:
 <a href="http://foo.com/bar.html#disqus_thread">Link</a>.
